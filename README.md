# AllTheBacteria

All bacterial isolate whole-genome sequencing data from INSDC, up to May 2025, uniformly assembled, quality-controlled, annotated, and searchable.

Follow up to Grace Blackwell's 661k dataset (which covered everything to Nov 2018).

Preprint: https://doi.org/10.1101/2024.03.08.584059

Website: https://allthebacteria.org

Documentation: https://allthebacteria.org/docs
